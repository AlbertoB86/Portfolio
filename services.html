App Escalada

- Proyecto Final de Ciclo -

Aplicación web desarrollada con Laravel y panel AdminLTE. Permite gestionar evaluaciones físicas, crear rutinas personalizadas, visualizar estadísticas con gráficas y localizar rocódromos mediante mapas interactivos con Leaflet.js.


Proyecto en proceso de mejora continua, orientado a incorporar nuevas funcionalidades y afianzar mis conocimientos en desarrollo web.

Pixel Palette

- Generador Paleta de colores -

Pixel Palette es una herramienta interactiva para generar paletas de colores con estética pixel art retro. Permite seleccionar armonías cromáticas, usar un selector visual o numérico en RGB/HEX, y aplicar colores a un diseño mediante drag & drop. Fue desarrollado pensado como una herramienta divertida y útil para diseñadores y desarrolladores.

To-Do List

- Práctica de Gestión de Tareas -

Aplicación web sencilla para crear, editar y eliminar listas de tareas. Desarrollada con HTML, CSS y JavaScript puro, pone el foco en la práctica de lógica de programación y manipulación del DOM sin frameworks. Utiliza LocalStorage para conservar los datos entre sesiones del navegador.

Página Web Blog Café

- Proyecto del Curso de Juan Pablo de la torre Valdez -

Este proyecto es una página web estática de práctica realizada como parte del curso “Desarrollo Web Completo” de Juan Pablo de la Torre Valdez.

Incluye: Maquetación con HTML5 y CSS3, Diseño responsive adaptable a móvil y escritorio, Optimización de imágenes con WebP y lazy loading, Formularios y navegación entre diferentes páginas.

Este portafolio ha sido creado con ilusión y ganas de aprender. Adaptado por mí a partir de una plantilla, sigue creciendo junto a mi formación como desarrollador.
